Lean 4 diagnostics for LaTeX PDFs

LeanTeX v2.5

Write Lean in native minted blocks, then print the matching Lean diagnostics or infoview-style state wherever the PDF needs it.

Download v2.5.0 View repository Open v2.5 PDF

Native minted first

v2.5 keeps the older LeanTeX v2 environment, but the recommended path is plain LaTeX: label a Lean minted block, configure the infoview mode, and call the output later. 

\usepackage[onefile]{leantexv2}

\leantexconfig{natcheck}{infoview=lines:4}
\begin{minted}[label=natcheck]{lean}
#check Nat.succ
example : 1 + 1 = 2 := by decide
\end{minted}

\leantexoutput{natcheck}
Output

Plain native display

Minted-backed output is rendered without an extra LeanTeX title bar or tcolorbox.

Control

Infoview modes

Use auto, full, goals, or lines:N.

Space

Lighter artifacts

Output-only snippets avoid duplicated generated code files.

Install

Clone
git clone https://github.com/rgrossharv/LeanTeX.git
cd LeanTeX
Pygments
python3 -m pip install Pygments
TeX files
PYTHONPATH=. python3 -m leantex install-tex
Build
PYTHONPATH=. python3 -m leantex build path/to/file.tex
