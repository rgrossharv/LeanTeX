LeanTeX Installation & Quick Start

LeanTeX lets you write Lean 4 blocks in LaTeX and render code + diagnostics into your PDF. This page is a single-file site for GitHub Pages.

1. Get LeanTeX

Clone the repo and enter the project folder.

git clone https://github.com/rgrossharv/LeanTeX.git
cd LeanTeX

2. Install Python Package

Normal install:

python3 -m pip install .

Dev/editable install:

python3 -m pip install -e .

If older pip errors on bdist_wheel:

python3 -m pip install --upgrade pip setuptools wheel

3. Install TeX Package

One-time install to your TeX tree:

leantex install-tex

This installs leantex.sty, leantexonefile.sty, and updates ~/.latexmkrc.

4. Minimal TeX Setup

\documentclass{article}
\usepackage[margin=0.5in]{geometry}
\usepackage{leantex}
\begin{document}
\begin{lean}
#check Nat.succ
\end{lean}
\end{document}

5. Build Commands

Direct build:

leantex build path/to/file.tex

Watch mode:

leantex watch path/to/file.tex

Module mode (without global install):

PYTHONPATH=. python3 -m leantex build path/to/file.tex

Troubleshooting

  • If PDF shows “Missing generated data”, run leantex build file.tex --no-latexmk first.
  • Ensure your file uses \usepackage[margin=0.5in]{geometry}, not reversed option syntax.
  • Run kpsewhich leantex.sty to confirm TeX can find the style.
  • If multiple hooks exist in ~/.latexmkrc, keep the LeanTeX one active.

GitHub Pages Setup (No Custom Domain)

  1. Create a new GitHub repo for this site.
  2. Drag and drop this index.html into the repo root.
  3. Go to Settings -> Pages.
  4. Source: Deploy from a branch, branch main, folder /root.
  5. Save and wait for publish.
